# Project Documentation

## Project Title
Firewall Simulator and Periodic Security Assessment System for Small Enterprises Using Java

## Project Overview
This project is a Java desktop application that simulates a firewall for a small enterprise. In this project, the small enterprise is represented by an educational institute. The system simulates virtual network devices, applies firewall rules, monitors network traffic, evaluates the security level, and generates a security report.

## Objectives
- Simulate a firewall system.
- Manage virtual network devices.
- Apply firewall rules.
- Monitor network traffic.
- Perform periodic security assessment.
- Generate security reports.

## Scope
The system includes:
- Administration devices.
- Student devices.
- Guest devices.
- Internal server.
- Printer.
- Internet connection.
- Firewall simulation.
- Security assessment.

## Technologies
- Java
- SQLite
- GitHub

## Database Design
The database includes the following tables:
- Users
- Devices
- Firewall_Rules
- Traffic_Log
- Security_Assessment

## System Components
- Main
- Device
- Firewall
- FirewallRule
- Traffic
- SecurityAssessment
- ReportGenerator

## Expected Output
The system will:
- Simulate network traffic.
- Apply firewall rules.
- Allow or deny connections.
- Calculate a security score.
- Display the risk level.
- Generate a security report.
